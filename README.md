# DataAnalytics_JobSimulation_Barclays
Data Analytics in partnership with Barclays (Springpod)
## Overview
A Virtual Experience as Barclays Data Analyst- Analyse a dataset of incidents reported by agents, draw conclusions, and present findings in a visual graph. 

## Objectives
- Learn key data analysis techniques
- Help improve operational efficiency

## Tools Used
- Tableau (sheet)

## Key Outcomes
- Identified data with trends from the shared raw data
- Translated findings into business recommendations
- Built a stacked bar chart to support decision-making in Tableau
- Updated analysis in the Workbook
- Uploaded project to the portal

## Skills Demonstrated
- Data Analysis
- Data Visualization
